# The Forgotten Endpoint

## Description
"The Forgotten Endpoint" is an interactive, fsociety-themed Capture The Flag (CTF) web application challenge. Agents are tasked with exploiting multiple vulnerabilities in order to retrieve 5 hidden flags. Good luck, agent.

## How to Start the Website

Ensure you have Node.js installed on your system.

1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Run the Application:**
   ```bash
   npm start
   ```

Once started, the application will be accessible locally via your web browser at `http://localhost:3000`.

## How to Stop the Website

To stop the running application, navigate back to the terminal where the server is executing and press:
`Ctrl + C` 

This will gracefully terminate the running Node.js process.
